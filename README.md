# Web Deck Starter

웹 프레젠테이션 스타터킷. Claude Code로 열어서 덱을 만들어달라고 하세요.

```bash
git clone https://github.com/jsk3342/web-deck-starter.git my-deck
cd my-deck
pnpm install
pnpm dev
```

`localhost:3000/example` 에서 예시 덱을 확인할 수 있습니다.

**새 덱 만들기**: `src/decks/<name>/slides.tsx` 파일을 만들면 `localhost:3000/<name>` 에서 자동으로 접근됩니다.

자세한 사용법은 `CLAUDE.md` 를 참고하세요 — AI가 읽고 실행하는 매뉴얼입니다.
